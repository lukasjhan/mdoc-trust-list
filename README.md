# Simple NestJS API for MDoc Trust List

This is a simple NestJS API for MDoc Trust List.

## Installation

```bash
pnpm install
```

## Running the app

```bash
pnpm run start
```

It will run on port 3000 by default. You can change it by setting the `PORT` environment variable.

## API documentation

You can access the API documentation at `http://localhost:3000/api` or if you set a custom port, then at `http://localhost:{PORT}/api`
